# CSS DEV

## Classless CSS

### Combine CSS attribues in HTML/JSX without editing CSS file

```jsx
<div className="f jcc m10 p06 lightskyblue">
  <div className="fsxl">CSS</div>
  <div className="bold">DEV</div>
</div>
```

### From CSS DEV library

```css
.f {
  display: flex;
}
.jcc {
  justify-content: center;
}
.m10 {
  margin: 1rem;
}
.p06 {
  padding: 0.6rem;
}
.lightskyblue {
  color: lightskyblue;
}
.fsxl {
  font-size: x-large;
}
.bold {
  font-weight: bold;
}
```
